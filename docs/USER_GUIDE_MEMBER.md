# Member User Guide

## Getting Started

### Sign Up
1. Go to application URL
2. Click "Sign up"
3. Enter email (@amalitech.com or @amalitechtraining.org)
4. Create password (8+ chars, mixed case, numbers, symbols)
5. Check email for verification code
6. Enter verification code
7. Sign in

### Sign In
1. Click "Sign in"
2. Enter email and password
3. You'll be redirected to dashboard

---

## Member Capabilities

### View Assigned Tasks
- Dashboard shows tasks assigned to you
- See task title, description, priority, status
- Filter by status

### Update Task Status
1. Find task in your list
2. Click status dropdown
3. Select new status:
   - OPEN: Not started
   - IN_PROGRESS: Currently working
   - COMPLETED: Finished
4. Status updates notify admin and other assigned users

---

## Email Notifications

You'll receive emails when:
- Task is assigned to you
- Task status is updated by others
- Task is closed by admin

---

## Best Practices

- Update status regularly
- Mark tasks IN_PROGRESS when starting
- Mark COMPLETED when finished
- Check email for new assignments

---

## Troubleshooting

**Cannot see tasks:**
- Verify you're signed in
- Check you have tasks assigned
- Refresh page

**Cannot update status:**
- Verify task is assigned to you
- Check network connection
- Try refreshing page

**Email not received:**
- Check spam folder
- Verify email address
- Contact admin
