# Admin User Guide

## Getting Started

### Sign Up
1. Go to application URL
2. Click "Sign up"
3. Enter email (@amalitech.com or @amalitechtraining.org)
4. Create password (8+ chars, mixed case, numbers, symbols)
5. Check email for verification code
6. Enter verification code
7. Sign in

### Sign In
1. Click "Sign in"
2. Enter email and password
3. You'll be redirected to dashboard

---

## Admin Capabilities

### Create Task
1. Click "Create Task"
2. Enter title (required)
3. Enter description (optional)
4. Select priority (HIGH, MEDIUM, LOW)
5. Click "Create"

### View All Tasks
- Dashboard shows all tasks in system
- Filter by status: OPEN, IN_PROGRESS, COMPLETED, CLOSED

### Assign Task
1. Find task in list
2. Click "Assign"
3. Enter user ID
4. Click "Assign Task"
5. User receives email notification

### Update Task
1. Click task to view details
2. Click "Edit"
3. Update title, description, or priority
4. Click "Save"

### Close Task
1. Find task in list
2. Click "Close Task"
3. Confirm action
4. All assigned users notified

---

## Best Practices

- Assign tasks to active users only
- Use clear, descriptive titles
- Set appropriate priority levels
- Close tasks when complete
- Monitor task status regularly

---

## Troubleshooting

**Cannot create task:**
- Verify you're signed in as admin
- Check network connection
- Refresh page and try again

**Email not received:**
- Check spam folder
- Verify email address
- Contact support

**Task assignment fails:**
- Verify user ID is correct
- Check user is not deactivated
- Ensure task not already assigned to user
